為什麼要用 Slack

Slack 就是取代 Facebook 的聊天群組，個別把慕哲人社用到的獨立出來。所以它的介面功用很基本，就是代替 Facebook Messenger 或 Line 做對話、溝通用。

好處有下列：

  • Android 和 iphone 都有 app ，而且會有通知。
  • 群組分明、清楚，不像在 Facebook 聊天群組雜亂無章
  • 有夜間不打擾功能，不易影響休息

要做這些調整：

  • 組織內討論正事儘量不要在 FB ，訊息來源會搞混
  • 檔案放在上面有限制，請不要放太大檔案，大檔可傳到 Drive 把連結貼上來。

基本介紹

畫面說明

如果是有三欄的狀況

看板說明

看板名是用「1個數字」+「 可能有底線 」 + 「看板名稱組成」。 數字是行政分組的分類，數字後有底線表示該組下的主要群組，無底線表示子群組。

  1. 不分組，如 1_general 就示讓大家討論任何不分組的事情
  2. 課程組，包括 2_course （課程組綜合討論）、2labor_workshop（勞動工作坊）
  3. 活動組，如 3_activity 、 3welcomenew2016 （給迎新用的）
  4. 行政組，如 4_admin
  5. 宣傳組，如 5_advocate

進階用法

關掉不需要的通知

進到你不用收取通知的板，點「設定小齒輪」->「Notification Preference …」

會出現以下這個畫面

  1. 簡單設法就勾「Mute this channel 」就好了
  2. 分別可以設電腦跟手機的設定，其中三個選項分別是：
    1. 任何訊息都通知
    2. 有人標註你的時候通知
    3. 完全不通知
